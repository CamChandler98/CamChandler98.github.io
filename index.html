HeartString

HeartString is an alternative take on social media, meant to invoke the feeling of connecting with a stranger somewhere unexpected, like on the bus or in line. Users make anonymous posts called "Hearts" and from the replies on these posts, users can choose one person to connect with. after connecting two users are no longer anonymous to each other and can see which hearts belong to their connections.

Built with CSS, HTML, React, Redux, JavaScript, Python, Flask, Flask-SQLalchemy, PostgreSQL Socket.io, AWS, and Styled-Components

Live: HeartString

GitHub Repo: HeartString

Party Lure

Party Lure is a social media app that allows users to find and make groups of like minded individuals to band together and complete activities together in the fictional but popular aquatic based mmo, "Abyss Adventures".

Built with CSS, HTML, React, Redux, JavaScript, Python, Flask, Flask-SQLalchemy, PostgreSQL, AWS, and Styled-Components.

Live: Party Lure

GitHub Repo: Party Lure

Elixr

Elixr is a web application that allows users to submit fantastical beverages, leave a review of their beverage experience , view other reviews and explore a profile page that includes your reviews and beverages. Elixr is inspired by Untappd, a website where users can leave reviews of their drinking experience including the drink and the venue

Built with CSS, HTML, React, Redux, JavaScript, Express, Sequelize, PostgreSQL, AWS, and Styled-Components.

Live: Elixr

GitHub Repo: Elixr

GoodGames

GoodGames is a web application that allows users to view a library of video games and organize a custom selection of games into shelves. Users will also be able to view reviews of each game as well as publish their own. This project is inspired by Goodreads, a website where individuals can search for books from an existing library, catalog books into custom shelves, and write & discuss reviews.

Built with CSS, HTML, JavaScript, Express, Sequelize, Pugjs, PostgreSQL.

Live: GoodGames

GitHub Repo: GoodGames
